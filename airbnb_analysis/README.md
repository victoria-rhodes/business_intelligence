\# Airbnb Market Analysis: Columbus vs New York  
  
\## Author  
Victoria Rhodes  
  
\## Project Overview  
This project analyzes Airbnb listing data to compare the Columbus, Ohio and New York City markets. The goal is to \[briefly describe what decisions your analysis could support\].  
  
\## Research Questions  
  
1. Has the number of listings increased or decreased in the past few years?  
2. Which neighborhoods have the most listings?  
3. How does the room type availability differ between cities?  
4. Are certain neighborhoods more popular to stay in (i.e. more reviews)?  
5. Do certain room types have similar annual availability?
